=== Description ==

Site universitaire de l'IUT Paris Descartes permettant, pour l'instant de consulter 
l'actualité, son emploi du temps, réserver une salle.

Ce qui le différencie du site de l'IUT est qu'il permet la réservation de salle. 
Et peut-être plus tard, un chat en ligne, pouvoir écouter de la musique, 
un profil pour partager des publications, photos, organiser des évènements etc..

=== How to use ===

Nous avons fourni le script de la BDD, veuillez l'executé afin de pouvoir utiliser pleinement les ressources du site.

En arrivant sur la page de connexion vous pouvez vous identifier avec  : login    : claudomir.campos@digitim.fr
																		 password : 123


Vous arrivez ensuite sur la page d'accueil où vous devrez pouvoir intuitivement 
trouver ce que vous voulez.
Mais nous serons vous prévenir si vous essayer de rentrez dans nos accès pas encore 
prêt.
